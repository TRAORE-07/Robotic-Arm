# Robotic-Arm
This robotic arm is an IOT (Internet Of Things) project, equipped with Advanced Sensors, Wireless Connectivity and Augmented Reality (AR).

1) Hardware Components:
- Breadboard
- Arduino UNO
- Robotic Arm
- 3 SG90 Servo Motors
- 3 MG995 Servo Motors
- 1 DHT sensor
- 1 DC power adapter 5V
- Frame
- 1 Ball
- Jumping Wires


2) Technology used:
- Arduino IDE
